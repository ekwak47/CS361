About the features of Diet Data

Welcome to the Diet Data page. This is an application where you can input data about you eating habits or for any sort of day-to-day tracking.

When visiting the Add New form, this is where you can add data for any daily activity. Simply edit the textbox and format the data as you please

For a more detailed display, click here to access table data. Table data is edited and deleted by using the buttons by the desired date. To add data, use the form below and input data.

The Settings will help users edit display configurations.
